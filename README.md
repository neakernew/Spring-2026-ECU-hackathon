# Rock Paper Scissors Game

A simple browser-based **Rock Paper Scissors** game built with **HTML, CSS, and JavaScript**. This project lets the player choose Rock, Paper, or Scissors, play against the computer, and keep track of the score.

## About the Project

I created this project to build a classic game in a way that helped me practice front-end web development skills. Since Rock Paper Scissors is a simple and familiar game, it gave me a good chance to focus on coding the game logic, connecting JavaScript to the page, and improving the overall look and feel of the website.

This project also gave me experience working with:
- user input through buttons
- random computer choices
- score tracking
- DOM manipulation
- layout and spacing in CSS
- basic animation and interactivity

## Features

- Play Rock, Paper, or Scissors against the computer
- Random computer decision each round
- Live result message after each match
- Score tracking for the player and computer
- Interactive button-based gameplay
- Custom styling with CSS
- Support for a custom logo/favicon

## Built With

- **HTML** for the structure
- **CSS** for styling and layout
- **JavaScript** for game logic and interactivity

## How the Game Works

The player selects one of the three options:
- Rock
- Paper
- Scissors

The computer randomly chooses one option as well. The game then compares both choices and decides the outcome.

### Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Matching choices result in a tie

## Installation / How to Run

1. Download or clone the project files.
2. Make sure `index.html`, `style.css`, and `main.js` are in the same folder.
3. Open `index.html` in your browser.
